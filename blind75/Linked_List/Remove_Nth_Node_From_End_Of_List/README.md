# Remove_Nth_Node_From_End_Of_List


## 解題思路：

1. 暴力解
    就是找到用題目給定的linked list 找到目前個數 與 要刪除的節點是哪個
    之後再前一個節點把next指標 指向 下下一個node就可以
    > Time Complexity : O(n)
    > Space Complexity : O(1)
