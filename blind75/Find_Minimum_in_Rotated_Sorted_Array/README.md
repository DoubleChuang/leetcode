# Find_Minimum_in_Rotated_Sorted_Array

## 解題思路：
看完題目講了一大堆 就是找最小值

1. sort解法
    就是重新sort後 回傳第一個值
    > Time Complexity : O(nlogn)
    > Space Complexity : O(1)
2. min
    直接找出最小值
    > Time Complexity : O(n)
    > Space Complexity : O(1)
3. 二元搜尋最小值
    > Time Complexity : O(logn)
    > Space Complexity : O(1)
4. 遞迴搜索
    參考：https://www.youtube.com/watch?v=P4r7mF1Jd50&ab_channel=HuaHua
    > Time Complexity : O(logn)
    > Space Complexity : O(1)
